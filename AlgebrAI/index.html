📐 Polinomios
  • Potencia: x^2 o x**2
  • Ej suma: 3x^2+2x-5 + x^2-4x+1
  • Ej producto: (x+3)(x-2)
  • Tip: Usa ^ para exponentes
🔢 Factorización
  • Dif. cuadrados: x^2-9
  • Trinomio: x^2+5x+6
  • Factor común: 4x^3-8x^2
  • Cubos: x^3+8
𝑥 Ecuaciones Lineales
  • Formato: ax + b = c
  • Ej: 2x + 5 = 15
  • Ej doble: 3x - 7 = 2x + 5
  • Tip: Variable: x minúscula
𝑥² Ecuaciones Cuadráticas
  • Forma: ax² + bx + c = 0
  • Ingresa: a, b, c por separado
  • Método: Fórmula general
  • Soporta: raíces complejas
⚙ Sistemas 2×2
  • Métodos: Sustitución, Igualación, Eliminación
  • Forma: a₁x + b₁y = c₁
  • Ingresa: coeficientes directamente
  • Detecta: sin solución / inf. soluciones
㏒ Logaritmos
  • Bases: 2, 3, 5, 10, e (ln)
  • Ej: log₁₀(1000) = 3
  • Ej: log₂(8) = 3
  • Propiedad: cambio de base ln(x)/ln(b)
〜 Trigonometría
  • Funciones: sin, cos, tan, sec, csc, cot
  • Pitágoras: a²+b²=c²
  • Conversión: grados ↔ radianes
  • Identidades: verificación numérica
📈 Gráficas
  • Potencia: x**2 (no x^2)
  • Trig: Math.sin(x), Math.cos(x)
  • Valor absoluto: Math.abs(x)
  • Raíz: Math.sqrt(x)
